# news

## Установка проекта
```
\client npm install
\server npm install
```

### Запуск клиента и сервера на хосте
```
\client npm run serve
\server npm run build
\server npm run start
```

### Сборка клиента в папку \client\dist
```
\client npm run build
```

### Сборка сервераив папку \server\build
```
\server npm run build
```
